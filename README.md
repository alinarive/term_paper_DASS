# Job Satisfaction in Russia (RLMS-HSE Analysis)

## Overview

This repository contains the code used for the empirical analysis of job satisfaction in Russia as part of an academic research project. The study investigates temporal, generational, and regional differences in job satisfaction using data from the RLMS-HSE survey (2002–2024).

The analysis includes regression models, dimensionality reduction techniques, clustering, network analysis, and machine learning methods.

## Repository Contents

* `final_job_satisfaction-2.ipynb` — main Jupyter Notebook containing data preprocessing, statistical analysis, modeling, and visualization.
* `README.md` — project description and documentation (this file).

## Data

The RLMS-HSE dataset used in this study is not included in this repository due to size.

The data is available via Google Drive:
[https://drive.google.com/your-link-here](https://drive.google.com/drive/folders/11Wri2g_i0TXmJIPQhcx45CbRqq8sbrNN?usp=sharing)

All preprocessing steps required to reproduce the results are implemented in the notebook.

## Methods

The following methods are applied in the analysis:

* Ordered logistic regression
* Principal Component Analysis (PCA)
* K-means clustering
* Network analysis
* Random Forest models

## Reproducibility

To reproduce the results:

1. Download the dataset from the provided Google Drive link.
2. Open `analysis.ipynb`.
3. Update the file paths to point to the local dataset location.
4. Run all cells sequentially.

All results, figures, and tables presented in the paper are generated directly from the notebook.
